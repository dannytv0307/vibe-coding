Start the backend dev server locally.

---

## Steps

1. Check if `backend/` directory exists
2. Run the dev server with watch mode:

```bash
cd backend && npm run start:dev
```

Server starts at **http://localhost:3000**

If `node_modules/` is missing, run `npm install` first, then `npm run start:dev`.

Report:

```
✅ Backend dev server running at http://localhost:3000
```
